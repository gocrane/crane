Join the Crane community

Crane is an open source project that anyone in the community can use, improve, and enjoy. We'd love you to join us! Here's a few ways to find out what's happening and get involved.

Learn and Connect

Using or want to use Crane? Find out more here:

  • Slack: Chat with other project users in #cloud-resource-analytics-and-economics
  • Community Meetings: Live discussion of new features and issues

Develop and Contribute

If you want to get more involved by contributing to Crane, join us here:

  • GitHub: Development takes place here!
  • Slack: Chat with other project developers
  • Community Meetings: Live discussion of new features and issues

You can find out how to contribute to these docs in our Contribution Guidelines
